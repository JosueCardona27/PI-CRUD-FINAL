# PI-CRUD-FINAL
PI CRUD Final es un proyecto web desarrollado como producto integrador, enfocado en la gestión completa de un sistema mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar). Durante el desarrollo de dicho proyecto mi trabajo principalmete fue de fronted, pero en ocasiones tuve que entrar al backend y añadir funcionalidades extras, añadidos del carrusel de turnos, logout de perfil, funcionalidad de orderdetails, mandar datos de las ordenes al carrusel, entre muchos otros, mas informacion de commits en el repositorio original https://github.com/NexusJC/PI-CRUD.git y mas detalles del proyecto en README que se encuentra dentro de la carpeta PI-FINAL
